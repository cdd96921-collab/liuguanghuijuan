# ✨ 流光绘卷 · 粒子画板 v6

一款基于 Web 的交互式粒子解压工具，让您在流光溢彩中释放压力、捕捉灵感。

---

## 📖 产品简介

「流光绘卷」是一款创新的情绪疗愈工具，通过粒子系统模拟流体绘画效果，让用户在数字画布上自由创作，获得身心放松的沉浸式体验。

**核心理念**：将情绪转化为可见的流光轨迹，通过视觉反馈实现情绪释放与自我觉察。

---

## ✨ 核心功能

### 🎨 创作模式
- **手动绘制**：按住鼠标拖动生成流光粒子轨迹
- **自动模式**：粒子随流场自由流动，被动观看放松
- **保留路径**：轨迹不再消散，永久留存创作成果
- **画笔预设**：细笔 / 中笔 / 粗笔 / 喷枪四种预设，一键切换
- **压感模拟**：根据鼠标移动速度自动调节笔触粗细（速度越快越细，范围 0.5–1.0）

### 🛠 画布工具
- **吸管取色**（I）：从画布拾取颜色作为粒子覆盖色
- **橡皮擦**（E）：擦除局部轨迹
- **文字标注**（T）：在画布上添加文字
- **画布缩放**：滚轮缩放（0.25–4 倍，以鼠标为中心）、平移工具拖动视图、`0` 键重置
- **图片导入**：导入图片作为新图层

### 🗂 图层系统
- 多图层创作（最多 8 层），每层独立离屏 canvas
- 拖尾与粒子绘制到活跃图层，主画布实时合成所有图层
- 图层增删 / 显隐 / 不透明度 / 重命名 / 排序 / 合并
- 图层状态随会话持久化

### 🎵 音频可视化
- 捕获电脑系统音频输出（不启用麦克风）
- **音频演示模式**：无需共享系统音频，用 Web Audio 合成三层波形（60Hz/440Hz/2kHz）+ LFO 调制，静音驱动粒子（设置菜单开关）
- **多频段节拍检测**：独立检测低音（鼓点）、中音（旋律）、高音（打击乐）节拍
- **频谱颜色映射**：14 色完整光谱，低频对应暖色调，高频对应冷色调
- **自适应 BPM**：自动估计音乐节奏，动态调整粒子响应频率
- **分层粒子生成**：低频生成大粒子、中频生成中粒子、高频生成小粒子
- **节拍爆发效果**：鼓点触发中心辐射、旋律触发彩色扩散、打击乐触发亮点飞溅
- 实时频谱指示器（BASS/MID/HIGH）+ 节拍时控制面板柔和高亮

### 🎭 主题与形状
- 5 套精美主题色板（彩虹、海洋、火焰、极光、糖果），每套包含 12-16 个颜色点，色彩过渡更丰富平滑
- **AI 自定义主题**：通过风格迁移生成专属调色板（见 AI 创作）
- 3 种粒子形状（圆形、星形、光芒）
- 主题切换时粒子平滑过渡

### 🤖 AI 创作
- **风格迁移**：上传图片，AI 分析色彩风格，生成专属调色板与粒子参数（主题、形状、大小、速度、密度、拖尾），一键套用
- **情绪识别**：输入心情文字，AI 识别情绪并推荐最匹配的主题（彩虹/海洋/火焰/极光/糖果）
- 基于阿里云百炼 qwen3.7-plus 大模型（OpenAI 兼容接口）
- 请求含 30s 超时与 1 次自动重试

### 📤 分享导出
- **分享卡片**：生成 1080×1080 分享图（作品 + 署名 + 日期 + 右下角 QR 码），可预览与下载
- **WebM 动画导出**：录制画布 10 秒动画，VP9 优先回退 VP8，录制时倒计时提示
- QR 码生成器内嵌于 share.js，无需第三方库，离线可用

### ✨ 灵感画廊与每日挑战
- **灵感画廊**：12 套预设作品（5 主题 × 3 形状的不同参数组合），缩略图首次打开时离屏 canvas 渲染生成，一键套用参数
- **每日挑战**：30 条创作提示按日期循环，启动时自动展示，设置菜单可随时查看

### 💾 作品管理
- 本地作品库（IndexedDB v2），保存作品缩略图、心情标签与**自定义标签**
- **标签筛选**：画廊顶部 tag chips 一键筛选
- **批量管理**：批量选择 / 全选 / 批量删除
- **画廊骨架屏**与空状态引导（首次使用可一键加载灵感预设）
- 情绪日记功能，记录创作时的心情状态
- 7 天心情趋势图表
- 支持 PNG 图片导出与 SVG 矢量导出
- 撤销 30 步 + 撤销历史预览网格

### 🎛 参数调节
- 粒子大小、速度、密度、拖尾长度
- 一键随机参数，快速探索不同效果

### 📱 移动端手势
- **双指轻点**：撤销（< 200ms 且无明显移动）
- **双指捏合**：缩放画布（以双指中点为中心）
- **单指长按**：吸管取色（600ms 且移动 < 8px，仅画笔工具）
- 多指同时绘制支持

### ♿ 无障碍设计
- 色盲模拟（红色盲、绿色盲、蓝色盲）
- 高对比度模式
- 完整键盘快捷键支持
- ARIA 无障碍标签
- 尊重 `prefers-reduced-motion`（减少初始粒子上限）

### 🎬 微交互
- 按钮点击涟漪扩散、保存成功闪白 + 粒子扩散
- 粒子计数实时监控（绿/橙/红三色预警）
- 滑块拖动数值放大预览
- 主题切换背景平滑过渡

## 🚀 快速开始

### 环境要求
- 现代浏览器（Chrome ≥ 88、Edge ≥ 88、Firefox ≥ 85、Safari ≥ 14）
- 必须通过 HTTP/HTTPS 服务器访问（ES Modules 在 file:// 协议下受 CORS 限制无法加载）
- 系统音频捕获另需 HTTPS 或 localhost 环境

### 安装与运行

**方法 1：本地服务器（推荐）**

### 模块化架构

项目采用 ES Modules 规范，10 个模块通过 `import/export` 建立清晰依赖关系，浏览器原生加载，零构建工具。

| 模块 | 职责 | 依赖 |
|------|------|------|
| `util.js` | 纯工具函数（lerp/rand/clamp/lerpHue/hexToRgba/escapeHTML/formatDate 等） | 无 |
| `particle.js` | Canvas 所有权、粒子系统、精灵图缓存、渲染循环、历史快照、FPS 监控 | util, main(state), audio, layers |
| `audio.js` | 系统音频捕获、频谱分析、节拍检测、音频驱动粒子生成、合成音效 | util, main(state), particle, ui |
| `storage.js` | IndexedDB 封装（artworks/diary 两个 store）、缩略图生成 | particle(canvas) |
| `layers.js` | 图层管理器（多离屏 canvas 合成、拖尾衰减、快照/恢复、增删/显隐/不透明度） | 无（纯自包含） |
| `inspirations.js` | 灵感画廊 12 套预设、30 条每日挑战、离屏 canvas 缩略图渲染（Map 缓存） | 无（纯自包含） |
| `ai.js` | AI 风格迁移（视觉）、情绪识别（文本）、OpenAI 兼容接口封装、30s 超时 + 1 次重试 | main(state), util |
| `share.js` | 分享卡片生成（1080×1080）、WebM 录制导出、内嵌 QR 码生成器（GF(256) + Reed-Solomon） | particle(canvas), ui(showToast) |
| `ui.js` | DOM 引用、图标系统、Toast、控件状态、模态框、画廊、日记、引导、图层面板、AI/分享弹窗 | util, main(state), particle, audio, storage, inspirations |
| `main.js` | state 单例、输入处理、编排函数、初始化序列、键盘快捷键、图层编排、AI/分享编排 | 全部 9 个模块 |

**循环依赖安全保证**：模块间存在循环引用（如 main↔particle、particle↔audio），所有跨模块绑定访问都在函数体内执行，不在顶层求值时读取，ES Module 的 live binding 语义确保函数调用时所有模块已完成求值。`layers.js` 无外部依赖，`particle → layers` 为单向依赖。

## 🔧 技术栈

- **前端框架**：原生 HTML5 / CSS3 / JavaScript (ES6+)
- **架构**：ES Modules（10 个独立模块，零构建工具，浏览器原生加载）
- **渲染引擎**：Canvas 2D API（主画布 + 多图层离屏 canvas 合成）
- **音频处理**：Web Audio API
  - `getDisplayMedia` 捕获系统音频
  - 演示模式：3 路振荡器（60Hz/440Hz/2kHz）+ LFO 调制合成伪频谱，静音驱动粒子
  - AnalyserNode 三频段（BASS/MID/HIGH）独立节拍检测 + 自适应 BPM
- **视频录制**：MediaRecorder API（`canvas.captureStream` → WebM，VP9 优先回退 VP8，4Mbps）
- **AI 集成**：阿里云百炼 qwen3.7-plus 大模型（OpenAI 兼容 /chat/completions 接口，含 30s 超时 + 1 次重试）
- **本地存储**：IndexedDB（artworks/diary 双 store，v2 带 byTag 索引）+ localStorage（state 子集持久化）
- **图标系统**：内联 SVG
- **二维码**：内嵌 QR 码生成器（GF(256) + Reed-Solomon + 8 掩码择优，零第三方依赖）

## 🐛 常见问题

### Q1：音频反应不工作？

**A**：请检查：
- 浏览器是否为 Chrome/Edge 桌面端（其他浏览器支持有限）
- 是否在 HTTPS 或 localhost 环境下运行
- 在共享面板中是否勾选了"分享系统音频"
- 电脑是否正在播放音频

### Q2：作品保存后找不到？

**A**：作品保存在浏览器本地 IndexedDB 中，不会上传到服务器。清除浏览器数据会导致作品丢失。建议同时下载 PNG 文件备份。

### Q3：移动端无法使用音频功能？

**A**：移动端浏览器通常不支持 `getDisplayMedia` 的音频捕获。可通过设置菜单 → "使用音频文件"上传本地音乐文件。

### Q4：粒子形状切换后不生效？

**A**：切换形状后需要拖动绘制才能看到新形状效果。形状会保存在精灵图缓存中，所有交互都会使用当前形状。

### Q5：页面卡顿怎么办？

**A**：
- 尝试降低粒子密度
- 降低拖尾长度
- 关闭音频反应
- 检查 FPS 监控（右上角），持续低帧会自动降级
